CM3028 - Web Application Development

Module Information

On completion of this module, students are expected to be able to:

  1. Recognise and discuss the Model 2 web application architecture and apply it to the design of web applications.
  2. Apply suitable technologies in the construction and integration of web application components.
  3. Use a Model 2 architecture to assemble web application components into a 3−tier web application of significant size.
  4. Appraise and implement appropriate security policies in a 3−tier web application.
